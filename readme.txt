FakeSkyBox creator v.1.0

Данная программа (BAT) предназначена для создания фальшивого скайбокса для 
использования их как браш текстура.

1. УСТАНОВКА
Создайте папку в директории игры рядом с папкой "materials" "materialsrc" и 
поместите туда программу.

2. СОЗДАНИЕ ТЕКСТУРЫ
Программа заточена для работы с готовыми vmt файлами. После открытии программы 
вам нужно вписать имя относительно корневой папки. При верном вводе процесс 
пройдет автоматически и ваша текстура будет готова. Возможно в конце вам нужно
будет нажать на любую клавишу для продолжения.
